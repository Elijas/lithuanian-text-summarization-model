Paruošt aplinką:

```bash
pip install pipenv
pipenv install
```

Paleisti:

```bash
pipenv run streamlit run streamlit_app.py
```

Paleisti tam tikram port'e:

```bash
python3 -m streamlit run streamlit_app.py --server.port 80
```
